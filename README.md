## Random Color Generator
A simple front-end project built with HTML, CSS, and JavaScript.
The app generates random hex colors every second and displays them on the screen. It has a start button to begin the color cycling and a stop button to pause it.

### What I practiced

- DOM manipulation (querySelector, style)
- setInterval and clearInterval for timing
- Generating random hex color codes using JavaScript
check it out [here](https://samy253.github.io/random-color-generator/)
